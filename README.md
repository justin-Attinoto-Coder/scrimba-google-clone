# Scrimba HTML & CSS Solo Projects

This repository contains two solo projects from the Scrimba HTML & CSS course:

## 1. Google Clone
A pixel-perfect recreation of Google's homepage featuring:
- Authentic navigation header with links and sign-in button
- Google logo (SVG)
- Functional search bar with icons
- Search buttons
- Language options
- Complete footer
- Fully responsive design

**Location:** Root directory (`index.html`, `styles.css`)

## 2. The Daily Dribble
A newsletter signup page for soccer news featuring:
- Hero image
- Email subscription form
- Call-to-action button
- Terms of service link
- Clean, centered layout

**Location:** `daily-dribble/` directory

## Getting Started
Install the dependencies and run the project:
```bash
npm install
npm start
```

Then open:
- Google Clone: `http://localhost:5173/`
- Daily Dribble: `http://localhost:5173/daily-dribble/`

Head over to https://vitejs.dev/ to learn more about configuring vite
